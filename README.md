# Dreamwear Backend (Ecommerce de Pijamas)

Este proyecto es un backend desarrollado en Spring Boot para gestionar el ecommerce Dreamwear.

## Modelo de Dominio (resumen)

- Producto
- Usuario
- Carrito
- CarritoItem
- Orden
- OrdenItem
- Categoria (opcional)

## Tech Stack

- Java 17
- Spring Boot 3
- Spring Data JPA
- H2 / MySQL
- Lombok
- Maven
